# 1-hour-test-assignment

RoR - React blog

### How to launch:
  1. git clone https://github.com/gladkiy98/1-hour-test-assignment.git
  2. cd server/
  3. $ bundle
  4. $ rake db:create
  5. $ rake db:migrate
  6. $ rails s

  --open new terminal--

  7. cd ../
  8. cd client/
  9. npm install
 10. npm start

### How to use:
  localhost:3001/
